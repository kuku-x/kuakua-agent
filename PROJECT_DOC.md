# 专属行为感知·AI夸夸Agent 软件项目文档

## 一、整体概述
本软件基于开源电脑行为监控项目ActivityWatch二次独立开发，搭建专属AI智能Agent。能够**自动记录用户电脑全天使用行为**，结合大模型进行数据分析、每日总结、个性化暖心夸赞、时间管理建议，并附带情绪陪伴聊天功能，实现真实化、个性化、不空洞的AI陪伴与正向鼓励。

## 二、详细完整功能（全部功能点）
### 1. 电脑全行为监控采集功能
1. 自动记录电脑所有软件使用时长、窗口打开记录
2. 统计网页浏览、各类应用使用频率
3. 区分工作学习类软件、娱乐休闲类软件
4. 本地持久化存储每日行为数据，不上传云端，保护用户隐私
5. 自动按日期划分每日数据，实现单日行为汇总

### 2. 每日行为智能总结功能
1. 自动读取当日全部电脑使用数据
2. 智能分析整体使用状态、专注时长、碎片化时间
3. 生成简洁清晰的**每日使用总结报告**
4. 梳理用户一天的时间分配情况

### 3. 个性化真实夸夸功能（核心主打）
1. **结合真实行为夸赞**，不编造、不油腻、不通用空洞彩虹屁
2. 根据用户努力、专注、自律、学习进度、时间规划进行针对性夸奖
3. 挖掘用户每日闪光点、坚持的地方、做得好的细节
4. 语气温柔治愈，偏向情绪正向激励
5. 夸赞内容贴合当天真实使用情况，一人一份专属总结夸赞

### 4. 时间管理&生活建议功能
1. 根据用户使用习惯、作息、软件使用频率给出**可落地建议**
2. 针对久坐、用眼、娱乐过多、专注不足等问题提出优化建议
3. 学习工作效率提升建议
4. 合理规划上网时间、劳逸结合建议

### 5. 智能陪伴聊天&情绪安慰功能
1. 独立对话聊天模块，支持用户自由输入内容倾诉
2. 情绪疏导、压力安慰、暖心鼓励
3. 日常对话、心情疏导、精神陪伴
4. 面对焦虑、疲惫、内耗进行正向安抚与心理支持

### 6. 本地桌面程序运行功能
1. 纯本地桌面应用，无需浏览器打开
2. 后台监控服务常驻，持续采集数据
3. 主程序一键启动运行
4. 数据全部存储在本地电脑，隐私安全

### 7. 项目工程化相关功能（你的Git项目自带）
1. 代码独立仓库管理，二次开发完全独立，与原开源项目无关联
2. 模块化代码结构：数据采集模块、AI分析模块、夸夸生成模块、聊天模块
3. 环境配置解耦，API密钥安全管理
4. 可扩展后续功能、迭代升级

## 三、软件特色亮点（写报告非常加分）
1. **真实数据驱动**：所有夸赞与建议均来自用户真实电脑行为，而非固定文案
2. **完全私有化本地运行**：数据本地存储，隐私安全性高
3. **模块化解耦设计**：底层监控与上层AIAgent分离，不修改原开源项目源码
4. **双重模式**：每日自动总结模式 + 自由聊天陪伴模式
5. **正向情绪价值**：主打鼓励、治愈、自我肯定、时间管理
6. 桌面exe程序，双击即可使用，无需额外环境依赖

## 四、极简版（适合项目简介、首页介绍）
本AI夸夸Agent桌面软件可以自动监控记录用户电脑日常使用行为，每日对用户的时间分配、专注情况进行智能总结，结合真实使用数据生成专属暖心夸赞与合理化改进建议，同时支持用户自由聊天、情绪倾诉与安慰陪伴，实现个性化AI正向陪伴。

## 五、超精简版（一句话功能总结）
能够监控电脑日常使用、自动生成每日行为总结、结合真实表现进行专属夸赞、给出生活学习建议，并具备聊天安慰、情绪陪伴功能的本地桌面AI智能Agent软件。
